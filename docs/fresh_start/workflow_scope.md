# MVP Workflow Scope (Weeks 1-6)

## Chosen Workflow

**Weekly customer re-engagement recommendations for ecommerce brands**.

The product generates personalized "next best action" recommendations for low-engagement customers and sends one approved message through a single outbound channel.

## Why this workflow

- High frequency: repeats every week.
- Easy business linkage: conversion and repeat revenue are measurable.
- Cross-functional relevance: CRM, lifecycle marketing, and support can all use it.
- Suitable for OpenClaw skills: clear sequence of data ingest -> recommend -> explain -> deliver -> capture feedback.

## Primary Persona

**Growth/CRM manager at a D2C ecommerce brand**.

## Must-Win Action Per Session

The manager accepts at least one recommendation and publishes at least one targeted outreach message.

## Scope Freeze (No Additions Until End of Week 6)

Included:
- One workflow only: low-engagement customer reactivation.
- One channel only: Telegram outbound message format.
- One recommendation cycle per week.
- Human approval required before delivery.
- Telemetry event capture for each step.

Excluded:
- Domain-specific analysis tooling outside reactivation.
- Multi-channel delivery (email, push, SMS, WhatsApp).
- Full campaign builder UI.
- Dynamic pricing or ticket inventory optimization.
- Any autonomous send without human approval.

## Week 6 Exit Criteria

- Workflow runs end-to-end with no manual coding during operation.
- Recommendation explanations are present and readable by non-technical users.
- At least one recommendation-to-action signal is collected each week.
